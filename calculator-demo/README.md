# Calculator Demo

Ein einfacher Java-Taschenrechner als Beispielprojekt.

## Features

- Addition, Subtraktion, Multiplikation, Division
- Fehlerbehandlung für Division durch Null
- Maven-Build

## Build & Run

```bash
mvn compile
mvn exec:java -Dexec.mainClass="com.example.calculator.Main"
```

## Tests

```bash
mvn test
```
